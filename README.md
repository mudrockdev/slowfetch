# Slowfetch

## Slowest fetch script ever made with intention

## To set default color

slowfetch set colorName

## Or just call it with color

slowfetch colorName

## You can reset the color with

slowfetch set reset

## Supported colors

"black", "red", "green", "yellow", "blue", "magenta", "cyan", "white", "gray".
